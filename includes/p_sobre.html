Tecnologias

Bootstrap

Framework HTML, CSS e JavaScript para desenvolvimento de interfaces web responsivas. Originalmente criado para a interface do Twitter, rapidamente tornou-se um dos mais populares frameworks de interface.

getBootstrap.com

Coverstrap

Templates para Boostrap que estendem o framework original, agregando componentes para a composição de páginas. Este plataforma é inspirada no template Lighter.

coverstrap.com

jQuery

Biblioteca JavaScript que facilita a manipulação de eventos dos navegadores, a implementação de comportamentos avançados, como movimentos, rolagens de página, animações e Ajax.

jQuery.com

PHP

Popular linguagem de script para desenvolvimento web. Nesta aplicação, a tecnologia é utilizada para organizar componentes da página e evitar duplicidade de código HTML.

php.net

Timeline JS

Ferramenta de código aberto desenvolvida pela equipe Northwestern University para a criação de timelines dinâmicas e interativas. Os dados são inseridos em uma planilha do GoogleDrive.

timeline.knightlab.com

Facebook Plugins

O Facebook disponibiliza uma série de plugins que podem ser inseridos em páginas web com funcionalidades como compartilhar, curir e comentar. Nesta aplicação são utilizados os dois últimos.

Social Plugins

Como utilizar?

Para utilizar esta plataforma, basta clicar no botão abaixo, salvar os arquivos no seu computador a abrir o arquivo index.php em seu navegador de internet. Para isso, no entanto, é preciso que esteja configurado um servidor web com PHP. O Xampp é um kit que facilita a implementação de servidores web em PCs.

Uma vez rodando a aplicação em seu computador, altere o menu em index.php e as páginas de conteúdo no diretório /includes
