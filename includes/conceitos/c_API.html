Acrônimo de Application Programming Interface ou, em português, Interface de Programação de Aplicativos. Serve para usufruir no seu site de funcionalidades de um determinado serviço web que roda em outro servidor. A API tem uma documentação que ensina o programador a fazer a requisição dos conteúdos ou comportamentos. Um exemplo de API é a do Google Maps. Se você possui em seu banco de dados uma lista de locais, como endereços de suas lojas, pode por meio da API criar automaticamente um mapa com os pontos. Outra API é a do YouTube, que permite exibir a lista de vídeos de determinado canal, por exemplo. Facebook e Twitter também possuem APIs, utilizadas em sites e aplicativos para dispositivos móveis.
