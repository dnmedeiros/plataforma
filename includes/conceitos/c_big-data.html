Big data é um termo comum utilizado para descrever o crescimento exponencial e disponibilidade de dados, estruturados ou não. Pode ser entendido também como conjuntos de dados extremamente grandes e que, por este motivo, necessitam de ferramentas especialmente preparadas para lidar com grandes volumes, de forma que toda e qualquer informação nestes meios possa ser encontrada, analisada e aproveitada em tempo hábil.

Segundo Doug Laney, pesquisador da Gartner, Big data pode ser definido em três Vs, volume, velocidade e variedade:

  • volume: quantidade grande de dados e que crescem exponencialmente. Milhares de transações são geradas por segundo no mundo todo, como transações comerciais online, posts em redes sociais, sensores e máquinas se comunicando. O desafio aqui é determinar a relevância destes dados gerados e armazenados e como usar ferramentas e técnicas de análise de dados para criar valor a partir destes dados.
  • velocidade: A geração de dados ocorre a uma velocidade muito grande, e o tratamento destes dados precisa ser feito de forma adequada e ágil, quase que em tempo real. Reagir rapidamente a esta velocidade é muito importante para a sustentabilidade do negócio, e se coloca como um desafio para muitas empresas. Exemplo: imagine se um sistema de cartão de crédito não conseguir analisar rapidamente todos os dados que podem indicar que uma transação se trata de uma fraude.
  • variedade: este aspecto diz respeito à diversidade de formatos em que os dados são gerados. Dados estruturados em banco de dados relacionais, informações criadas por aplicações de negócios. E, dados não estruturados oriundos de documentos de texto, email, vídeo, áudio, mídias sociais e assim por diante. Gerenciar, juntar e lidar com esta grande variedade de dados como parte de um todo é um desafio, já que um tipo de dado pode ser inútil se não for associado a outros.
