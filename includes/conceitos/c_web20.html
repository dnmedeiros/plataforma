O termo Web 2.0 é utilizado para representar uma nova filosofia adotada por desenvolvedores e usuários de internet. Em sua essência, estão funcionalidades colaborativas. Os sites deixam de ser compostos por páginas criadas por um editor e passam a ser constantemente modificadas pela comunidade. Por isso, no Brasil o termo também é conhecido como web colaborativa.

Produtos da Web 2.0 são blogs, wikis, sistemas baseados em folksonomias, plataformas para armazenamento de fotos, vídeos, etc. Esses produtos tornam-se melhores quanto mais os usuários os utilizam.

O termo Web 2.0 é atribuído ao editor Tim O'Reilly.
